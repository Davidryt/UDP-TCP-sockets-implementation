Simple echo TCP client and servers.


In this directory:

* Cliente_TCP.c - Cliente TCP

* Cliente_UDP.c - Cliente UDP

* Servidor_TCP.c - Servidor TCP

* atdate.c - main


To build:

make clean; make


To run:

* cliente TCP

./atdate -h host -p port -m ct

cliente UDP

./atdate -h host -m cu

servidor TCP 

./atdate -m s
